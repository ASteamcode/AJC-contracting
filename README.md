## 🚀 Deployment

### Option 1 — GitHub Pages (Free, recommended)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/ajc-contracting`
5. Add your custom domain (e.g. `www.ajc-contracting.com`) in Pages settings

### Option 2 — Netlify (Free, custom domain)
1. Connect this GitHub repo to Netlify
2. Deploy settings: build command = none, publish directory = `/`
3. Add custom domain in Netlify dashboard

### Option 3 — Any web host
Upload all files to your hosting provider's `public_html` folder via FTP.

## ✏️ How to Update Content

All content is in plain HTML. Open any file in a text editor and search for the text you want to change.

### Update contact info
Search for `+961 1 000 000` and replace with your real phone number.

### Update office address
Search for `Metn, Lebanon` to find all address instances.

### Update email
Search for `Ajccontracting@gmail.com` — appears in all pages.

### Update working hours
Search for `7:30 AM` to find all hours instances.

### Replace placeholder photos
All photos use Unsplash URLs. Replace with your own:
```html
<img src="YOUR_PHOTO_URL" alt="description"/>
```

### Add your real logo
Replace the text logo in the nav with an `<img>` tag pointing to your logo file.

## 🔍 SEO

The site is optimized for Lebanese search traffic:

- **Meta tags** — title, description, keywords in EN/AR/FR
- **Schema markup** — `GeneralContractor` JSON-LD on homepage
- **Service schemas** — `Service` JSON-LD on each service page
- **Geo tags** — `LB-JL` / Metn, Mount Lebanon
- **hreflang** — EN, AR, FR language signals
- **Sitemap** — submit `sitemap.xml` to Google Search Console
- **Robots.txt** — crawler instructions

### Submit to Google Search Console
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add your domain
3. Submit `https://yourdomain.com/sitemap.xml`
